Serving My Own Index File

Update the html-webpack-plugin

To serve a custom html file, the html-webpack-plugin needs some details.

  • Add an object to the parenthesis of the HtmlWPPlugin in the webpack.config.js
  • give the newly created object a key of template with a value of "./src/index.html"
  • give the newly created object another key of filename with a value of "./index.html"

    • The config file should look something like...

    const HtmlWPPlugin = require('html-webpack-plugin');
    module.exports = {
      plugins: [
        new HtmlWPPlugin({
          template: "./src/index.html"
        })
     ]
    };

    What's it do

    According to the html-webpack-plugin docs...

  • template gives this plugin the path to the template. Our template, our bespoke index.html file, will sit be located at ./src/index.html

    • Create an index.html file

    Make one && store it in the src directory.

  • Give it a recognizable document title
  • Include in the body a div with an id of "app"

    • TEST IT!

    Open A terminal if one is not open. cd into the project directory if the terminal is not already. Run...

    npm start

    Here, our hand-written index file is now served.
