Serving My Own Index File

Update the html-webpack-plugin

To serve a custom html file, the html-webpack-plugin needs some details.

  • Import html-webpack-plugin at the top of the file && save it to a variable. I'm calling this HtmlWPPlugin
  • Setup the file to export an object.
  • In the object, create a plugins key.
  • Set the value of the plugins key to be an array.
  • In the plugins array value, set an element in the array to be new HtmlWPPlugin().

    • The config file should look something like...

    const HtmlWPPlugin = require('html-webpack-plugin');
    module.exports = {
      plugins: [new HtmlWPPlugin()]
    };

    TEST IT!

    Open A terminal if one is not open. cd into the project directory if the terminal is not already. Run...

    npm start

    Here, an index.html that is auto-generate by Webpack-dev-server is displayed! The title of the page 'Webpack App' is a giveaway that the ui is auto-generated.
